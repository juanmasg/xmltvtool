# xmltvtool
Set of utilities to manage XMLTV files
```
Usage of ./xmltvtool:
  -d	Demo data
  -g	Check data gaps
  -j	Concatenate multiple XMLTV files
  -o	Check data overlaps
  -r	Read multiple XMLTV files

  -w	Watch for program schedules

  -now
    	Airing now

-s	Search programme
  -season string
    	Search this season
  -episode string
    	Search this episode
  -subtitle string
    	Search this subtitle
  -title string
    	Search this title
```
